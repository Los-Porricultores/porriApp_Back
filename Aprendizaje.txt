en lo que se refiere a la sintaxis. La cosa es un poco más complicada cuando se 
empiezan a mezclar conceptos con entornos virtuales, paquetes, etc.

Descripción: Este tío tiene cursos muy interesantes en muchos lenguajes, va desde

aunque se puede hacer en un fin de semana facilmente si se sabe programar previamente.

2. Django

Para esta parte tendremos que extrapolar los tutoriales a nuestro caso de estudio, ya
que hay algunos que son para web, o para mezclarse con otros tipos de frameworks, etc.

Titulo: Django Official Tutorial
Descripción: Es el típico tutorial de inicio de cualquier lenguaje de programación 
que suele basarse más en enseñarte las características que en explicar algo.
¿Por que lo pongo primero? Por que viendo otros tutoriales me he dado cuenta que
necesitas primero haber hecho algun tutorial de inicio de este tipo para que entiendas
al menos de manera aproximada de que te hablan en los tutoriales que se encuentran 
enfocados más a nuestro objetivo. Tras una breve descripción está el tutorial.

Si no sabeis como configurar inicialmente el proyecto para el tutorial podeis ver los 
10 primeros minutos del tutorial "Django and React Tutorial // 1 - Creating a 
Fullstack Application" y os quedará más claro como se hace.

	https://docs.djangoproject.com/en/3.0/intro/overview/

Titulo: Django Rest Framework Tutorial
Descripcion: este ya va más enfocado a lo que nosotros necesitamos, empezamos por
el quickstart y luego al tutorial que está dividido en 6 secciones. 

	https://www.django-rest-framework.org/tutorial/quickstart/

Titulo: Django and React Tutorial // 1 - Creating a Fullstack Application
Descripción:

	https://www.youtube.com/watch?v=uZgRbnIsgrA&list=FLZeHWirMt8y6FYOlg9ISk8g&index=2&t=0s

Titulo: Create Instagram Like App in Python Django
Descripcion: Este tutorial es lo más similar que he encontrado hasta ahora de lo que vamos a 
necesitar hacer. Mas que nada por que emplea herramientas de entornos virtuales, servidores y
cosas muy interesantes para poder programar todos con un entorno común. Para poder hacer 
este tutorial podeis loguearos con vuestra cuenta normal de hotmail o cualquiera de las que 
te dejan utilizar dandoles los permisos que os piden.

	https://www.education-ecosystem.com/tylersavery/R0kJ7-create-instagram-like-app-in-python-django/z1G44-creating-an-instagram-service-in-django/
